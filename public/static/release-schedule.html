Planned Release: Summer 2013.

  • Class Leaderboards. Right now, we only have quiz leaderboards.

Instructor:

  • Fine-tuning Question Designer: All header fields with pull down. Better default startups and rendering sample questions. Better layout---though self-enlarging is nice, too.

  • Improved management of student lists: Email reminders (to laggards) and class authentication beyond shared passwords. Basics are already in the system.
  • Declutter file manager view by checking and unchecking boxes (backup, deleted).
